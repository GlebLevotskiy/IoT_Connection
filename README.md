# IoT_Connection

### Проект нацелен на создание модулей регистрации IoT устройства, симуляции IoT устройства и перехвата сообщений на платформе Microsoft Azure IoT Suite.

### Использованные технологии:

* .Net Framework 4.8
* Microsoft.Azure.*
